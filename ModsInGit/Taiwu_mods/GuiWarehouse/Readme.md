1.2.1
1）增加按住Ctrl键隐藏分类和搜索界面
2）增加搜索功能，支持多条件搜索：      &符合表示并且      | 符号表示或者
	例一：“图纸” 搜索：图纸
	例二：“内功：&宝物” 搜索：带有内功属性的宝物  （搜索装备增加属性的时候要加上冒号：，不然会把内功秘籍也搜索出来）
	例三：“音律|乐器”：搜索带有音律或者乐器的物品
	例四：“四品&绝技|四品&身法|一品&乐器” 搜索：四品的绝技或者四品的身法或者一品的乐器
3）最后缩小了分类按钮之间的间隔
