# 太吾游戏变速

## 更新历史
### 1.0.1
- [new] 0.1x~16x游戏变速（初始设置8x）
- [new] 可配置变速开关按键（初始设置为N）
- [new] 扩展功能选项：死斗开启，读书开启，促织捕捉开启自动暂停变速（可随时按变速快捷键开启变速）
- [new] 任何键盘按键按下期间暂停变速（解决加速时大地图键盘移动失控等问题）
