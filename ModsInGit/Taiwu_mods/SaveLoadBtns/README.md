# SaveLoadBtns - 改变 S/L 制度的按钮
太吾绘卷的MOD插件。

在游戏画面的上方加入两个按钮，方便随时〝储存〞或〝载入〞游戏进行。


## 使用注意
* 本功能专为完美主义者(强迫症)而设，过度使用:有損身心!

* 请避免在任务中途储存，否则载入时不能出现在原来位置
