# UseStorageBook
> By sth4nothing  


修习仓库中的书  
可在选书界面筛选书籍，包括仓库背包、真传手抄、品级、门派等  
依赖**GuiBaseUI** [https://github.com/phorcys/Taiwu_Mods/tree/master/GuiBaseUI/]
